Basic Color Theory Terms

There's a lot of color theory jargon that can get a bit confusing. Below are the absolute basics to get you started in any visual-design-related converstaion. Many experienced designers still need to regularly refresh on some of these terms.

The Color Wheel

A circle composed of different hues and can be broken down into primary, secondary, and tertiary color grops.

Hue

Hue refers to the basic color found on the color wheel like red, yellow, blue. Hue is synonymous to color or pigment.

Tint

A hue when white is added to it.

Shade

A hue when gray is added to it.

Saturation

Refers to the intensity or boldness of a color depending on the light that it is in. The more saturated a color, the bolder it is.

Basic Color Combinations

When in doubt, visual designers can always rely on the color pairings below. These are the most basic ways in which people develop color palettes. Some common color pairings include monochromatic, complimentary, split complimentary, analogous, triadic, and tetradic. We musn't forget the most basic of color groups derived from the color wheel: primary, secondary, and tertiary colors. Click on each box to learn more about each color grouping.

Primary Colors

Red, yellow, and blue.

Secondary Colors

Purple, green, and orange.

Tertiary Colors

The combination of primary and secondary colors closest to each other on the color wheel.

Monochromatic

One color and the shade and tints derived from it.

Complimentary Colors

Colors located opposite of each other on the color wheel.

Split Complimentary

Using the two adjacent colors to one's compliment.

Analogous

The use of colors located next to each other on the color wheel. This combination can potentially provide little contrast.

Triadic Colors

Three colors that are equal distance from each other on the color wheel..

Tetradic

Complimentary pairs of colors.

Accessibility in Design

The design of everyday things has become more and more democratized due to technology. These days, visual design reaches all kinds of people because a lot of it resides in the screens that surround us everyday. Even for signage and wayfinding, it is important to consider prevelant visual impairments that hinder people from effectively decoding the design. Here we get a quick overview of somethings to keep in mind when striving for visual accessibility.

Color

When it comes to picking colors, it is important to be intentional when choosing which to use. It is important to consider color blindness and potential eye strain whrn creating color palettes. A useful rule of thumb is to aim for high contrast in color pairing. Complimentary colors usually offer high contrast, while an analogous color scheme provides colors that are subtly different. In addition, colors should not be the solution to a design problem and should be accompanied by other indicators like text and icons. There are many contrast-checkers on the web to check your designs. For information on colors in accessibility, visit Smash Magazine.

Type

When choosing typefaces and fonts, there are a few simple ways to ensure the best legibility. First, use simple fonts for large bodies of text that aren't too complex or that have similar looking letters. For example, opt for a simple san-serif font over a display font for a paragraph and make sure that letters such as e and c are easily distinguishable. Second, make sure that there is enough contrast between the text and the background. If the contrast is too faint, it will be difficult to see; if there is too much contrast, it can cause the text to seem like its vibrating. For more information about choosing typefaces visit Web Aim.
